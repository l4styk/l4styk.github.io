About

I had an established career, but it required me to work weekends and holidays. I wanted to have more quality time with my family. My background was environmental science and I wanted to stay in a STEM field. I have two girls and I want them to know that they can do anything and be anything. For me, leading by example was the right choice. I began to talk with my friends that were in the technology field. This peaked my interest and seemed like it could be a good fit. So, while I was out on maternity leave I began to teach myself HTML and CSS with online courses. I was hooked! I enrolled in Portland Code School’s JavaScript class to accelerate my learning. Becoming a Web Developer has been a great fit for me. I like that I am challenged every day. That I continue to learn. That there will always be something new to discover. I am so happy to be a part of this exciting and ever changing field.

Project R

www.projectr.herokuapp.com/

Project R is a group project that was created as a capstone for Portland Code School. Done in JavaScript, it uses Backbone and Express. I was responsible for the full screen button and creating the Backbone views.

Graph Alchemist

www.graphalchemist.github.io/Alchemy/#/docs

Working under contract for GraphAlchemist, I was responsible for updating the docs website of AlchemyJS. It is in CoffeeScript.I used Grunt-Markdown to convert the markdown pages to HTML. I then used AngularJS to create the different views. The sidebar is dynamic and uses the header tags on the page to insert the correct words. This allows the documentation to be edited without having to update the sidebar.
